class hopwise.model.knowledge_aware_recommender.cafe.CAFE(config, dataset)¶

Bases: hopwise.model.abstract_recommender.KnowledgeRecommender

CAFE is a knowledge-aware recommender system that uses symbolic reasoning over a knowledge graph to explain recommendations.

Note

Assumes that each relation corresponds to a unique pair of entity types. e.g. ui-relation -> (user, item)
